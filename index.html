Peel this page! You heard me, peel it, just like an orange

The inside truth:

  • This is a JQuery Plugin. Nuff Said!
  • Have you seen JQuery UI ? Its awesome and this plugin is awesome. We go in pairs.
  • Its inspired/based upon the super sexy curl plugin by Elliott Kember
  • The original plugin didn't allow moving the peel effect to corners other than the "top-left" and "top-right". This one does!

  • I heart CSS3. Hence, This plugin uses just one image and the awesome CSS3 Transform : Rotate

    Net Effect : This wont work in your shitty browsers! Boo! Hiss!

  • For all you lazy bums, I added "click" and "hover" actions to peel this page on its own.
  • You can only add 1 and only 1 peel per page. For the sake of the sanity of the author of the original plugin and myself, dont go crazy peeeeling everything.

Receipe

  • First, take a HTML container with a target element like:
  • Add some JQuery and the JQuery UI : 
    <script src="http://ajax.googleapis.com/ajax/libs/jquery/1.4.2/jquery.min.js"></script>
<script>!window.jQuery && document.write('<script src="js/jquery-1.4.2.min.js"><\/script>')</script>
<script src="http://ajax.googleapis.com/ajax/libs/jqueryui/1.8.5/jquery-ui.min.js"></script>
<script>!window.jQuery.ui && document.write('<script src="js/jquery-ui-1.8.5.custom.min.js"><\/script>')</script>
  • To make JQuery UI edible, put some JQuery UI theme sauce : 
    <link rel="stylesheet" href="css/ui-lightness/jquery-ui-1.8.5.custom.css">
  • Now that the base is ready, lets go ahead and add some peel dressing. 
    <script src="js/turn.js"></script>
<link rel="stylesheet" href="css/turn.css">
  • Finally, heat it once the pan is ready 
    $(document).ready(function(){
  $('#target').fold({
    side:'bottom-left', //['top-left','top-right','bottom-left','bottom-right] Default: 'top-left'
    action:'click',    // ['click','hover'] Default : 'drag'
    startingWidth:40,   // starting width and height of the peel over image
    startingHeight:40   // default: 100px
  });
});

Any doubts ?

  • Which make of the JQuery sauce ?

    Grab the latest available. The newer the better!

  • Where to put the base element ?

    The plugin wraps the element to peel with an "absolute" positioned element. So it will take a corner as per the closest ancestor which is "absolute" or "relative" positioned.
